Deploying Applications with Kubernetes

Recently, I have had the exciting opportunity to try Kubernetes as an on-premise deployment solution. Kubernetes is a tool to orchestrate deployments made using containerization software such as Docker. It gives these containers a place to live (these are called nodes) and abstracts away the challenge of managing IP addresses, scaling, and the other details of making deployments work.
